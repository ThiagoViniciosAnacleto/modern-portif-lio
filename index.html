Landing Page Para Escritório

Escritório Advocacia

Este projeto consiste em uma landing page moderna, responsiva e altamente funcional, desenvolvida para um escritório de advocacia localizado em Curitiba, Paraná. O objetivo principal da página é atrair novos clientes, apresentar os serviços jurídicos oferecidos e facilitar o contato direto por meio de um formulário de consulta gratuita. A estrutura da página foi pensada para ser leve, acessível e visualmente atraente, utilizando Tailwind CSS, ícones modernos, integração com Google Maps e suporte a vídeo institucional. Toda a experiência do usuário é focada na confiança, clareza e credibilidade, características essenciais para o setor jurídico.

Sistema de Chamados PowerDesk

PowerDesk

O PowerDesk é um sistema web de suporte técnico desenvolvido para atender às necessidades internas de empresas que lidam com atendimento, manutenção e registros de chamados. O sistema foi projetado com foco em organização, agilidade e controle de demandas técnicas. Desenvolvido em Python (Flask) com banco de dados PostgreSQL, o PowerDesk permite o cadastro, acompanhamento e edição de chamados técnicos com registro automático de alterações (log), controle de permissões por tipo de usuário (administrador, técnico e comum) e uma interface padrão!
